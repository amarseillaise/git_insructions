## Азы git
****


**Создание репозитория**
* git init
****


**Выбор активного файла**
* git add

методы:
1. "." - выбирает все файлы в папке
2. "file_name" - выбирает конкретный файл
****


**Сохранение версии файла (фиксация)**
* git commit

методы:
1. "-a" - сохранение всех файлов в папке 
2. "-m" - добавление комментария
